# OutSourcedEmployee Payment System (Java)

Projeto em **Java** para cálculo de pagamento de funcionários terceirizados, aplicando **Programação Orientada a Objetos** com **herança** e **polimorfismo**.

## Conceitos utilizados
- Herança (`OutSourcedEmployee` → `Employee`)
- Polimorfismo (lista `List<Employee>`)
- Encapsulamento
- Sobrescrita de métodos (`payment()`)

## Funcionalidade
- Funcionários comuns
- Funcionários terceirizados com taxa adicional
- Cálculo automático de pagamento
